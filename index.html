Data Analysis - Farmers Market

This project is to understand the Farmer's Market growth and distribution in the US. I have analyzed the data and prepared a detail report on the key findings. All the plots are constructed using R libraries such as dplyr, ggplot2 etc..

Model to Predict Mental Health of an Employee

In this project I have used Data Mining algorithms to develop a model which predicts mental health of an employee based on the survey data that is collected from 2014. You model is as good as your data and data cleaning was the most important task I have performed in order to feed the model. The variables in the dataset collected each year varies and understanding which variable is vital for our study I have eliminated certain columns. I have built Logistic Regression and KNN algorithms and compared various model metrics using Adj-R2 and RMSE value for selecting the best model

Design of Relational Database System

This project focuses on design and development of Relational database using MySQL. I have devised logical model using MS Visio and Normalized the database to 3NF to eliminate update, delete and insert anomalies. I have also created Stored Procedures and Views to increase the effeciency of data retrieval from the databse. Further, to understand the graph database system I have replicated the Airline database on Neo4j using Nodes and Relationship and retrieved data using Cypher queries (NoSQL).

Design of Data Warehouse

This project is build a Data Warehose with 10 million records from 3 different datasets to understand the Impact of Weather and Covid on Accidents rate in the United States. The first step of this project is to identify data quality issues that need to handled before moving the data to warehouse. We have used Talend for data profiling and designed dimensional modelling in Star Schema. We created 3 different staging areas which helps us in data cleaning and error handling using SQL Server Management Studio and built ETL pipelines using SSIS, SSAS, and SQL Server. For advanced analytical purpose, we have created OLAP cube using business logic and understanding. The cube enabled us to slice and dice the data in multidimension and data marts and created to process Pre-Covid and Post-Covid data.
